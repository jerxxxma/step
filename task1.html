Есть 3 способа добавить CSS в HTML
  • Добавить css в HTML можно тремя способами,первый способ это создать файл с расширением .css и прописывать в нём все свойства и значения.
  • Второй способ это прописать в HTML, элемент <style> и в нём прописывать все свойства css.
  • Третий способ добавить css в html, это напрямую прописывать стили в элементе к которому хотим применить css.
  • В данном задании использовались свойства background-color: которое задаёт цвет фона и color: которое меняет цвет шрифта.
